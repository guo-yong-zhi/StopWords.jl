StopWords.stopwords — Constant

stopwords is a lazy dictionary of stop words for multiple languages. You can access the stop words for a given language using the language's name or ISO 639 code. For example, to get the stop words for English, you can use stopwords["eng"], stopwords["en"], or stopwords["English"].

Base.haskey — Method
haskey(sw::StopWordsDict, lang::String) -> Bool

Check if lang is a supported language. You can use either the language's name or its ISO 639 code. For example, haskey(stopwords, "en"), haskey(stopwords, "eng"), and haskey(stopwords, "English") all return true.
